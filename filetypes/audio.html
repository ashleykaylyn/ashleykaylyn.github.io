Audio Files

M4A

  • Audio-compressed file used for storage
  • iTunes use .m4a instead of mp3
  • Tend to be higher quality

FLAC

  • Free Lossless Audio Codec
  • The compression can save file space
  • A higher definition MP3

MP3

  • Mepg audio layer 3 file format
  • Compression saves valuable space
  • Very popular for mobile audio

MP4

  • Mistaken as the improved version of the MP3 file
  • MP4 file types are comprehensive media extensions
  • Contains data in the file, rather than code

WAV

  • Waveform Audio File that stores waveform data
  • The data stored presents an image that shows
    the strength of the volume. Such as an ocean wave
